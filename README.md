# flames-relationship-oracle
FLAMES (Friends, Love, Affection, Marriage, Enemies, Siblings) is a fun relationship prediction game.  This project features: - ✨ Glassmorphism UI - 🎡 Animated FLAMES Wheel - 🎉 Confetti Effects - 🌌 Particle Background - 📋 Copy Result - 🔗 Share Result - 📱 Responsive Design  Built using HTML, CSS, and JavaScript.
