# 🔥 FLAMES Relationship Oracle

A modern and interactive FLAMES (Friends, Love, Affection, Marriage, Enemies, Siblings) relationship prediction game built using HTML, CSS, and JavaScript.

## 🌐 Live Demo

https://raghunandan-22.github.io/flames-relationship-oracle/

---

## ✨ Features

* 🎡 Animated FLAMES Wheel
* ✨ Glassmorphism User Interface
* 🎉 Confetti Celebration Effects
* 🌌 Animated Particle Background
* 📋 Copy Result to Clipboard
* 🔗 Share Result Feature
* 📱 Fully Responsive Design
* ⚡ Fast and Lightweight

---

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

---

## 🚀 How It Works

1. Enter your name.
2. Enter your friend's/crush's name.
3. Click the "Calculate FLAMES" button.
4. The animated wheel spins.
5. The final relationship result is displayed.

Possible outcomes:

| Letter | Meaning   |
| ------ | --------- |
| F      | Friends   |
| L      | Love      |
| A      | Affection |
| M      | Marriage  |
| E      | Enemies   |
| S      | Siblings  |
